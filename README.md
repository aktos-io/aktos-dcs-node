# aktos-dcs-node

Node.js port of [`aktos-dcs`](https://github.com/aktos-io/aktos-dcs) library (v2).

# Additional Features 

`aktos-dcs-node` provides following transport and connectors:

### Transports 

* Serial Port Transport 

### Connectors 

* CouchDB
* SocketIO (Server + Client)
* TCP (Server + Client)
* Omron
  * Hostlink 
  * FINS 
* Siemens 
  * S7 Comm
* Raspberry 
  * Digital Input 
  * Digital Output
